# Subagents (project-level)

Project-level subagents for cursor-scaffold. For bootstrapped projects, the same layout is created with a short README pointing here and to the knowledge-hub.

- **Templates and global setup**: See the knowledge-hub repo `.cursor/agents/` (verifier, test-runner, debugger, documenter) and `.cursor/notes/cursor-subagents.md`. Copy to `~/.cursor/agents/` for use across all projects.
- **Define as needed**: Like helpers—when repeatable delegation would help, add a `.md` file here (or in a project's `.cursor/agents/`) with clear YAML frontmatter and description so Agent delegates automatically.
